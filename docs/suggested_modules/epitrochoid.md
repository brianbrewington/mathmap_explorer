# Epitrochoids (Spirograph)

## Motivation

An epitrochoid is the curve traced by a point on (or inside/outside) a
small circle rolling around the outside of a larger fixed circle. This is
exactly the mathematics behind the Spirograph toy — and it produces an
extraordinary range of beautiful looping patterns from just three
parameters. The patterns range from simple ellipses to intricate
multi-lobed flowers, all governed by the ratio of the two radii.

Epitrochoids extend MathMap's parametric curve collection into the world
of roulette curves — curves generated by rolling. They connect to gear
theory, cycloid mechanics, and Fourier analysis (every epitrochoid is
equivalent to the sum of two circular motions, i.e., two Fourier terms).

## Mathematical Background

A circle of radius r rolls without slipping around the outside of a
fixed circle of radius R. A pen is attached at distance d from the center
of the rolling circle.

```
x(t) = (R + r)·cos(t) − d·cos((R + r)·t / r)
y(t) = (R + r)·sin(t) − d·sin((R + r)·t / r)
```

Special cases:

```
d = r  → epicycloid (cusps touch the fixed circle)
d < r  → curtate epitrochoid (loops don't cross the fixed circle)
d > r  → prolate epitrochoid (loops extend beyond)
R = r  → limaçon family (including the cardioid when d = r = R)
```

The curve closes when R/r is rational. If R/r = p/q (in lowest terms):

```
Period = 2π · q
Number of loops = p
```

The total arc length for one period of the epicycloid (d = r):

```
L = 8r(R + r) / R    (per cusp segment)
```

Fourier connection — the epitrochoid can be written as:

```
z(t) = (R + r)·e^(it) − d·e^(i(R+r)t/r)
```

This is the sum of two rotating complex exponentials — exactly two terms
of a Fourier series. Every Spirograph pattern is a two-frequency
harmonic!

## Connections

- **Foundations:** `lissajous` (also two-frequency parametric curves —
  Lissajous uses Cartesian frequencies, epitrochoids use polar/rolling),
  `rose-curves` (another family of polar parametric curves with
  integer-ratio symmetry)
- **Extensions:** `cycloid` (the special case of a point on the rim of
  a circle rolling on a straight line — the R → ∞ limit),
  `fourier-synthesis` (epitrochoids as two-term Fourier sums)

### Presets

- **Cardioid** — R=r=3, d=3 — heart-shaped single cusp (epicycloid with R=r)
- **Trefoil** — R=3, r=1, d=1 — three-cusped epicycloid
- **Spirograph classic** — R=5, r=3, d=2.5 — the quintessential looping pattern
- **Star** — R=5, r=1, d=1 — five-pointed star (epicycloid)
- **Nearly closed** — R=5, r=3.1, d=3 — almost repeats, slow precession
- **Limaçon** — R=3, r=3, d=4 — prolate limaçon with inner loop

### Interaction

Click and drag on the canvas to adjust the pen offset d in real time,
stretching or shrinking the loops. The rolling circle animation shows the
geometric construction — the user sees the mechanism that generates the
curve.

### Buttons

- **Reset** — return to default R=5, r=3, d=3
- **Redraw** — restart the curve animation from t = 0

## Implementation

### Rendering tier

**Tier 2 (2D Canvas)** — parametric curve drawing with optional rolling
circle animation overlay.

### Algorithm

Each animation frame:
1. Advance t by dt · drawSpeed
2. Compute position:
   - x = (R+r)·cos(t) − d·cos((R+r)·t/r)
   - y = (R+r)·sin(t) − d·sin((R+r)·t/r)
3. Append to trail buffer
4. If "show construction":
   - Draw the fixed circle (radius R, centered at origin)
   - Rolling circle center at ((R+r)·cos(t), (R+r)·sin(t))
   - Draw rolling circle, pen arm, and pen dot
5. Draw the trail curve
6. Check if curve has closed (t ≈ 2π·q for R/r = p/q)

Animating the pen offset d morphs the curve continuously from cusped
epicycloids (d = r) through curtate and prolate forms. Animating r
while keeping R fixed changes the number of loops, with clean integer
ratios producing closed curves that snap into symmetric patterns.

## What the User Learns

Complex beauty from simple mechanics. A pen, a wheel, and a table — that's
all a Spirograph is, and that's all an epitrochoid is. The user discovers
that the visual complexity of the pattern is controlled by a *ratio*:
R/r = p/q means the curve has p loops and closes after q revolutions. When
the ratio is irrational, the curve never closes, precessing forever. This
is the same rational-vs-irrational dichotomy that appears in rose curves,
Lissajous figures, and musical intervals. The Fourier connection is the
deeper insight: every Spirograph pattern is literally two sine waves
added together in the complex plane, making it a physical embodiment of
harmonic synthesis.
